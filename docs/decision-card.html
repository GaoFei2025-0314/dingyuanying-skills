看透事 · 七步拆解

丁元英

透过现象看规律，按客观规律办事，为自己的因果负责，不等救世主。

自
度
  1. 一剥层

    剥到文化属性

    技术 → 制度 → 文化，连剥三层。表面是招法，往下是规则与利益格局，最底下是反复生产这个结果的思维方式。根子几乎总在第三层。

    问自己

    是什么思维方式，在反复生产这个结果？

  2. 二自检

    强弱势文化自检

    弱势文化是一个"靠"字：靠别人、靠运气、靠"应该如此"。强势文化是按事情本来的规律动作。只对自己的念头用，不给别人贴标签。

    问自己

    这个念头里，有多少在"靠"，有多少在"动"？

  3. 三鉴路

    见路不走

    别人走通的路，是他在他的特定条件下走通的。学规律，不学表相；照抄之前先做条件差异审计。

    问自己

    他成功时的关键条件，我缺哪几条？缺的是不是决定性的？

  4. 四认道

    认客观规律

    神即道，道法自然。市场、人性、时间、概率——不因你的愿望而改变。把力气全押在可改变的变量上，停止跟规律较劲。

    问自己

    哪些是我改不了、只能顺应的？我还在跟哪条规律较劲？

  5. 五因果

    不昧因果

    种什么因，得什么果，不打折、不能赊账。想要果却不想种因，就是"破格获取"，早晚要还。

    问自己

    我要的果，对应的因是什么？那个因，我愿不愿意、能不能种？

  6. 六自度

    自性自度

    得救之道，救主并不存在。贵人、平台、风口、"等我准备好"——都是等救主。承认没有人会来，才可能开始自己动。

    问自己

    我在等谁来救？不等了——此刻我能自己动的第一步是什么？

  7. 七放下

    无所住而生其心

    做事时全力，做完不执着于得失。把能控制的行动和控制不了的结果分开——尽人事，让下一个动作依然清醒。

    问自己

    哪些是我能控制的动作？哪些是控制不了的结果？我的焦虑押在了哪边？

红线学他的方法，别学他的孤傲；用它的清醒，补它的温度。不拿"弱势文化"贴人标签，不把"我只是说事实"当刻薄的许可证。

安顿心 · 自在五问

芮小丹

自性自在，不昧因果。不表演、不怨、不占有、不把自己押在外部，每个当下按本心行事。

自
在
  1. 一观众

    去掉观众

    把决定里的"观众"找出来：父母、同学、朋友圈、社会时钟。表演是最贵的成本——用一生取悦观众，观众却随时散场。

    问自己

    如果这件事永远没有人知道，我还会这么选吗？

  2. 二卸载

    卸掉"应该"

    怨气 = 现实与"应该"的差额。把"他应该、公司应该、努力应该"逐条改写成"实际是"。不是认命，是把力气花在真实的棋盘上。

    问自己

    我在要求世界按我的剧本来吗？实际是什么样的？

  3. 三当下

    回到当下

    焦虑总是活在过去（悔）或未来（怕）。该干什么干什么——本分事不需要先解决人生全局，本分本身就是定心的锚。

    问自己

    此刻、今天，我的本分是什么？

  4. 四去执

    爱而不占有

    抓取的信号：记账式付出、改造工程、用对方的回应给自己打分。爱是"我看见你、我成全你"；占有是"我需要你来填我"——填不满的洞在自己身上。

    问自己

    我的付出，是让他更成为他自己，还是换取他属于我？

  5. 五收根

    拆掉抵押

    如果答案是"什么都不剩"，问题不在那件事，在于你把整个自己抵押了出去。把价值的根收回来——根在自己这儿，外面的得失就只是天气。

    问自己

    没有这份工作 / 这段关系 / 这个头衔，我还是谁？

红线学她的干净，别拿她的完美苛责自己；学她的自在，别忘了她脚下有地基。"不占有"不是单方面讨好，"卸掉应该"不是劝人认命。

接住机会 · 五步法

肖亚文

普通人的机会方法论。顺序即因果：前四步是年功，第五步是秒功——没有前四步，"敢接"就是赌博。

敢
接
  1. 一眼力

    识货

    "认识这个人就是一扇窗户，仅此而已。"识货在先，图报在后甚至不图报。眼力是专业的副产品——看不懂的领域谈不上识货。

    问自己

    我关注的人和事，是因为热闹，还是因为值钱？

  2. 二年功

    长线

    低频、长期、有分寸：不谄媚、不刷存在感、不透支。人脉不是你认识谁，是谁认可你——维系的资本是专业与可靠，不是殷勤。

    问自己

    这段关系五年不产生任何"用处"，我能不能接受？

  3. 三记录

    本分

    机会的分配逻辑是托付：关键时刻，人们把事交给"历史记录干净"的人。你的每一件小事都在写这份记录。可靠 = 能力 × 分寸。

    问自己

    我过去两年的记录，能让人放心把大事交给我吗？

  4. 四核算

    冷判

    恐慌和狂热都不是信息。去一手信息处求证，别在二手情绪里打转。把机会拆成三格：最好情况、最坏情况、兜不兜得住。

    问自己

    最好情况是什么？最坏情况是什么？最坏情况我兜得住吗？

  5. 五下注

    敢接

    机会的窗口极短，且总是化装成风险出现。敢接的底气来自冷判，不来自胆子——真看懂的值得重注，看不懂的多小都不碰。

    问自己

    如果失败，我能不能说清楚为什么当时值得做？

红线学她的年功和冷静，别只记住她惊险的一跳——跳板搭了很多年，那一跳才成立。功利化是这套方法的失效条件；下注永远先问"输光了我在哪儿"。

事情复杂用丁元英，心里拧巴用芮小丹，机会当前用肖亚文——没有救世主，尽人事而不执于果；根在自己这儿，得失只是天气；跳板搭了很多年，那一跳才成立。
