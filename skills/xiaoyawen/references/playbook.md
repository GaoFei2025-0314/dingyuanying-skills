# 肖亚文五步法 · 现代职场打法

四个高频场景，每个都标出对应的步骤。核心口诀：**识货、长线、本分、冷判、敢接——顺序即因果。**

---

## 场景一：怎么跟"层次比我高的人"建立关系（第一、二步）

**常见误区**：要么不敢接近（"人家凭什么理我"），要么用力过猛（请客、吹捧、刷存在感）——两者都是把关系当交易。

**肖亚文的打法**：
1. **先识货再接近**。想清楚这个人身上什么值钱：是判断力、是视野、还是他所在的位置能看到的信息。说不清"值钱在哪"就先别经营,你维系不好看不懂的人。
2. **以工作交集为入口，用专业说话**。她与丁元英的关系起点是把助理工作做得无可挑剔。没有交集就创造轻交集：高质量的请教（做足功课、问题具体、别问搜索引擎能答的）、有价值的信息回流（"您上次提的 X，我看到一个相关数据"）。
3. **低频且有分寸**。一年几次有内容的接触，胜过每周点赞。不索取、不越界、不传话。
4. **接受长期不变现**。"认识这个人就是一扇窗户，仅此而已"——把收益预期降到"多知道些事"，关系反而立得住。

**本周动作**：列出你环境里最值得学习的 3 个人；为其中 1 个准备一次做足功课的请教。

## 场景二：跳槽 / 转型评估（第四步为主）

**常见误区**：因为情绪跳（受气了、烦了）或因为数字跳（只看涨薪 30%），都是让情绪和单一变量替自己冷判。

**肖亚文式冷判清单**：
- **最好情况**：这个新位置三年后能给我什么（能力、位置、窗口）？值不值得要？
- **最坏情况**：业务黄了 / 老板走了 / 承诺不兑现，我落在哪儿？简历上这段经历还成不成立？
- **兜底核算**：最坏情况发生，我的现金流撑几个月？回原赛道的门还在不在？
- **一手信息**：找到 2 个在那家公司/那个行业的真人聊过没有？只看 JD 和 HR 的话等于只看招股书宣传页。
- **对照第三步**：我现在的"托付记录"攒够了吗？跳槽最好发生在你**刚打完一场胜仗**时，而不是狼狈时——同样的你，标价完全不同。

## 场景三：有人拉我入伙 / 副业机会（第四、五步）

这是最接近她入股格律诗的场景。

**冷判**：
- 这生意的因果链成不成立（用 dingyuanying 的因果检验：为谁解决什么问题、凭什么是我们）？
- 拉我的人要的是我的**钱**、我的**能力**、还是只是需要**有人分担风险**？三者性质完全不同。
- 我投入的上限是多少（钱、时间、机会成本）？最坏全损，我兜得住吗？
- 我有没有**一手信息**：看过账吗？见过真实客户吗？还是只听了故事？

**敢接的判据**：
- 我能不能不看对方的 PPT，自己把这件事为什么成立讲一遍？讲不出来 = 没看懂 = 不碰，无论多小。
- 看懂了，就按"最坏可承受"定注码，然后不半途撤资（叶晓明式的中途退股，两头亏）。
- 记住她的结构：**用职员的积蓄换股东的位置**——最好的机会是身份跃迁的机会，不只是多一份收入。

## 场景四：被裁 / 停滞，感觉"没机会"（第一、三步补课）

**先破一个执念**："没机会"多数时候是**还没进入任何人的托付名单**。机会不是招聘网站上的帖子，是"出事了/要干大事了，第一个想到你"。

**补课顺序**：
1. **盘点托付记录（第三步）**：过去两年，谁会为我的哪件事背书？如果列不出 3 条，当前要务不是"找机会"，是去把 1-2 件事做出可背书的结果——哪怕在现在这份不满意的工作里。
2. **重新识货（第一步）**：我这个行业未来三年什么技能会涨价？我现有的哪块经验放到哪个相邻领域是稀缺的？
3. **激活长线（第二步）**：把旧同事、旧上级、旧客户的关系捡起来——不是求职群发，而是一对一、有内容的重新连接。多数好机会走的是这条路，不是简历海投。
4. **等窗口（第五步）**：窗口出现时（往往化装成"某处出了麻烦、缺个能顶上的人"），前面的功课决定你是不是那个被想起来的名字。

---

## 通用对话骨架

1. **定位**："你的问题在五步里的第几步？"（多数人卡在三，焦虑在五）
2. **拆情绪**："把'感觉'放一边——最好、最坏、兜不兜得住，三个格子填一下。"
3. **查一手**："这件事你有几分信息是一手的？"
4. **给动作**："本周能做的最小动作是 ___。"
5. **校准预期**："前四步是年功，第五步是秒功。别用赌的方式补课。"
