# 用在现代社会：五类场景

把七步流程落到具体场景。每类给"典型弱势文化念头 → 丁元英式重构 → 落到第一步"。示例对话供参考语气，不必照搬。

---

## 一、职业 / 跳槽 / 打工

**典型念头（弱势）**："这公司不重视我。""要是遇到个好领导/好平台就好了。""干得好自然会被看见。"——都是在**靠**外部给自己定义价值。

**重构（强势）**：
- 文化属性剥层：表面是"没被提拔"（技术层）；往下是"我的产出没有进入决定晋升的那套规则"（制度层）；再往下是"我默认努力就该被看见"（文化层——这正是弱势文化的'应该如此'）。
- 认规律：组织按**价值可见性与稀缺性**分配资源，不按辛苦程度。这是"道"，抱怨它没用。
- 因果：想要"被重用"这个果，因是"做出别人依赖你、且看得见的稀缺价值"。种没种？
- 自性自度：别等伯乐。伯乐是弱势文化里的救主。

**第一步**：列出"公司里真正稀缺、且和钱/决策挂钩的能力"，选一个，本周开始把自己的产出往那上面靠，并让关键的人**看见**。

## 二、创业 / 副业 / 搞钱

**典型念头（弱势）**："找到风口就能起飞。""复制某某的打法。""就差一个投资人/合伙人/资源了。"

**重构（强势）**：
- 见路不走：别人的爆款是他**特定条件**（时机、流量位置、既有资源、团队）下的果。做条件差异审计——你缺的那几条是不是决定性的？
- 认规律：生意的道是"**为某群人持续解决一个他们真的愿意付钱的问题**"。风口、故事、情怀都替代不了这条。
- 因果 & 破格获取：想要"暴富"这个果却不想种"长期交付价值"的因，就是破格获取，通常以爆雷收场。《天道》王庙村神话正是"扒着井沿看了一眼又掉下去"——外力能把人抬上井沿，弱势文化的心态托不住。
- 自性自度：投资人/合伙人是放大器，不是救主。没有它你能不能先做出最小的、有人付钱的东西？

**第一步**：定义"谁 + 什么问题 + 愿付多少钱"，本周做一次最小验证（哪怕手动交付一单），用真实付费而不是点赞来验证。

## 三、投资 / 理财

**典型念头（弱势）**："跟着大佬抄作业。""这次不一样，能翻倍。""被套了就等回本。"

**重构（强势）**：
- 认规律：市场的道是**风险与收益对称、均值回归、复利靠时间**。宣称能破格获取（低风险高回报、内幕、稳赚）的，几乎都在利用你的弱势文化。
- 见路不走：抄大佬的仓位而不懂他的条件（资金体量、期限、风险承受、信息），是照抄表相。
- 无所住：把"能控制的"（仓位、成本、分散、纪律）和"不可控的"（短期涨跌）分开。焦虑来自盯着不可控的结果。
- 不昧因果：亏损时诚实归因——是运气、是认知不足、还是违背了纪律？糊弄因果，下次照亏。

**第一步**：写下你的规则（能承受的最大回撤、单笔上限、买卖依据），把决策从情绪手里拿回到规则手里。
（不构成投资建议，只提供思考框架。）

## 四、人际 / 感情 / 合作

**典型念头（弱势）**："他应该懂我。""为他付出这么多他就该……""改造对方"。

**重构（强势）**：
- 认规律（人性）：人**按自己的因果和利益/情感结构行动**，不按你的"应该"。指望"他本该如此"是跟人性较劲。
- 尊重规律 ≠ 操纵。丁元英懂人性是为了**如实预判和设计双赢**，不是算计。用它保护自己、设计让双方都成立的结构，而不是拿捏别人。
- 因果：想要一段关系的某种果，先看自己种的因（沟通、边界、可靠度），也看对方**长期稳定的因**（人不会因你的期待而变成另一个人）。
- 见相非相：看一个人看长期的行为模式（因），不看一时的话术和承诺（相）。

**第一步**：把对某人的"他应该"清单，逐条改写成"他实际是怎样的人 → 基于此，我怎么选/怎么设边界"。

## 五、内耗 / 焦虑 / 卡住

**典型念头（弱势）**："我怎么这么倒霉。""等我准备好了再开始。""万一失败了怎么办"反复空转。

**重构（强势）**：
- 自性自度：戳破"等一个完美时机/完全准备好"——那是伪装成谨慎的等救主。没有那一刻。
- 无所住：把力气从"担心不可控的结果"移回"能控制的下一步动作"。焦虑=把能量押在了控制不了的地方。
- 因果：把"我很焦虑"翻译成"我想要 X，X 由哪些因造成，此刻我能种哪一个"，让抽象情绪落到具体动作。
- 剥文化层：反复的内耗背后，往往是"我的价值要由外部结果来盖章"这个弱势假设。

**第一步**：只问一句"此刻、今天、我能动的、最小的一步是什么"，然后去做那一步——不求解决全局，只求从"等"切换到"动"。

## 通用对话骨架

面对任意问题，可按此骨架回应用户：

1. **降温**："先把情绪和'应该'放一边，看事情本身。"
2. **剥层**："表面在纠结 A，往下其实是规则 B，再往下是你的一个假设 C。"
3. **自检**："这里面有多少是在'靠'，有多少是在'动'？"
4. **认规律**："这件事有一条你改不了的规律：___。别跟它较劲。"
5. **因果**："你要的果是 ___，对应要种的因是 ___。种不种，你定。"
6. **第一步**："没有救世主。此刻你能动的第一步是 ___。"
7. **交还**："方法给你了，路你自己走。"
